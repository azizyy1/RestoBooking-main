# RestoBooking

RestoBooking est une application web de réservation de tables pour restaurant, développée avec ASP.NET MVC.  
Le projet permet aux clients de réserver une table en ligne selon la date, l’heure, le nombre de personnes, le type de table et l’occasion choisie.

## Objectif du projet

L’objectif de ce projet est de simplifier la gestion des réservations dans un restaurant en proposant une plateforme claire, rapide et pratique pour les clients comme pour l’administrateur.

## Fonctionnalités principales

- Création d’une réservation en ligne
- Choix de la date, de l’heure et du nombre de personnes
- Sélection du type de table
- Calcul automatique du prix selon la catégorie de table et l’occasion
- Vérification de la disponibilité des tables
- Confirmation de réservation
- Annulation avec frais de 5%
- Interface administrateur pour consulter, modifier ou annuler les réservations

## Technologies utilisées

- ASP.NET MVC
- C#
- Entity Framework
- SQLite
- HTML / CSS
- Bootstrap

## Règles de réservation

Les réservations sont disponibles entre 13h et 23h.  
Le prix varie selon le type de table choisi et l’occasion sélectionnée.  
En cas d’annulation, des frais de 5% sont appliqués.

## Rôle administrateur

L’administrateur peut gérer l’ensemble des réservations depuis une interface dédiée.  
Il peut consulter les réservations, modifier certaines informations et annuler une réservation si nécessaire.
